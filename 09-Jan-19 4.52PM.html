Git and Bash Commands I've learnt today

Bash Commands:

  1. cd - Change Directory
  2. ls - List Directories
  3. open - Opens File in the current directory

Git Commands:

  1. git clone [URL]
  2. git status
  3. git add [FILE_NAME]
  4. git commit -m "message"
  5. git push
  6. git pull
